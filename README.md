CLASSIFICADOR DE TOSSE 


1. DESCRIÇÃO DO PROJETO

O Classificador de Tosse é um aplicativo Android desenvolvido
para analisar sons de tosse através de Inteligência Artificial.

O sistema utiliza um modelo de Machine Learning treinado com
amostras reais de áudio respiratório para identificar padrões
compatíveis com:

- Pneumonia
- Bronquite
- Saudável
- Outras condições respiratórias

O aplicativo foi criado para fins acadêmicos, investigação e
apoio preliminar, não substituindo avaliação médica profissional.


2. REQUISITOS DE SISTEMA


Versão mínima recomendada:

- Android 8.0 (Oreo) ou superior

Requisitos adicionais:

- Microfone funcional
- Espaço livre mínimo: 100 MB
- Permissão para gravação de áudio
- Memória RAM recomendada: 2 GB ou superior


3. INSTALAÇÃO DO APK


Caso o aplicativo seja instalado manualmente através do ficheiro
APK, o Android poderá bloquear a instalação por segurança.

Siga os passos:

PASSO 1:
Abra:
Configurações > Segurança

PASSO 2:
Procure a opção:

"Instalar apps desconhecidos"
ou
"Fontes desconhecidas"

PASSO 3:
Permita instalação para:

- Chrome
ou
- Gestor de ficheiros

PASSO 4:
Abra o ficheiro APK e clique em INSTALAR.


4. COMO UTILIZAR


MÉTODO 1 - GRAVAR TOSSE

1. Abra o aplicativo
2. Clique em "GRAVAR"
3. Tussa próximo ao telefone
4. Clique em "PARAR"
5. Aguarde resultado

MÉTODO 2 - IMPORTAR ÁUDIO

1. Clique em "CARREGAR ÁUDIO"
2. Selecione um ficheiro .wav
3. Aguarde processamento
4. Veja o resultado


5. RESULTADOS POSSÍVEIS


SAUDÁVEL
Nenhum sinal compatível detectado.

PNEUMONIA
Padrões compatíveis identificados.

BRONQUITE
Padrões compatíveis identificados.

OUTRA CONDIÇÃO
Não compatível com pneumonia ou bronquite.
Recomenda-se avaliação médica.
